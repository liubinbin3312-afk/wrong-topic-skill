# 📚 小学生错题管理助手

> 自动识别试卷错题、生成同类型练习、追踪学习进度

## ✨ 功能特点

| 功能 | 说明 |
|------|------|
| 🖼️ 错题识别 | 上传试卷图片，自动识别并提取错题 |
| 🧠 记忆存储 | 按学科、年级永久保存错题 |
| 📚 分类整理 | 按学科、年级、知识点自动分类管理 |
| ✍️ 同类型题 | 针对薄弱知识点生成强化练习题 |
| 🏆 智能掌握 | 连续答对5题自动标记为已掌握 |
| 👤 个性设置 | 支持昵称、头像，打造专属学习空间 |

## 🔄 工作流程

`
上传试卷 → 图片校验 → 识别错题 → 知识点分析
    ↓
生成练习 → 保存错题 → 复习提升 → 掌握解锁
`

## 🎯 核心机制

### 智能掌握判定
- 同一知识点连续答对 **5道题** → 自动标记为已掌握
- 答错时进度归零，从头累计

### 个性化学习
- 支持自定义昵称和头像
- 学习报告展示个人进度
- 鼓励性语言增强学习动力

## 📁 文件结构

`
wrong-topic-skill/
├── SKILL.md      # 技能定义文件
└── README.md     # 说明文档
`

## 🚀 使用方法

1. 将 SKILL.md 文件复制到 MiniMax Agent 的 skills 目录
2. 重启 Agent
3. 开始使用错题管理功能

## 📝 输出标签

- <错题识别> - 识别出的错题列表
- <知识点分析> - 知识点错误分析
- <同类型题> - 强化练习题
- <提升建议> - 学习改进建议
